This code controls the four pipes.

There are four pipe configurations and some other configurations for tuning / debugging.

CONFIG_PIPE1:

Configuration containing the notes for pipe 1, the right most pipe.

CONFIG_PIPE2:

Configuration containing the notes for pipe 2, the middle.

CONFIG_PIPE3:

Configuration containing the notes for pipe 3, the left pipe that is not the bass.

CONFIG_PIPE_BASS

Configuration for the bass pipe.

Each configuration contains the notes for its respective pipe and also changes other settings as well.

The other settings are as follow:

LED_WHEN_STEP:

Turns on the LED while the pipe is moving

SPEAKER:

Turns on the speaker and it will play the note over the speaker as well

MANUAL_INPUT:

Input steps manually as opposed to using MIDI values
